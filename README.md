# 🛍️ TrendyThreads – Fashion Website

## 📌 Overview

TrendyThreads is a modern fashion website designed for a local clothing business.
It showcases stylish products for men and women and allows customers to place orders directly through WhatsApp.

This project focuses on clean UI, smooth user experience, and simple ordering without complex payment systems.

---

## ✨ Features

* 🧥 Browse products by categories (Men / Women / New)
* 🎯 Filter products by type (Dress, Hoodie, Shirt, etc.)
* 🔍 Product detail popup with description
* 📱 Order directly via WhatsApp
* 🛒 Simple and clean product UI
* 💖 Responsive and modern design
* 🏷️ Custom logo and favicon branding

---

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript
* Canva (for logo & design assets)

---

## 📸 How It Works

1. User visits the website
2. Browses products using categories and filters
3. Clicks on a product to view details
4. Clicks **Order Now**
5. WhatsApp opens with pre-filled message
6. User sends message to place order
